% ---------------- ABSTRACT ----------------
\section*{Abstract}

Customer churn is a critical challenge for businesses, as retaining existing customers is more cost-effective than acquiring new ones.

This project presents an AI-driven customer analytics system that evolves from predictive analytics (Milestone 1) to an agentic AI-based retention strategy (Milestone 2).

In Milestone 1, the system predicts churn using classical machine learning models, identifies key churn drivers, and provides interactive insights through a Streamlit application.

In Milestone 2, the system is enhanced with Generative AI and Agentic Intelligence, enabling it to explain predictions and recommend personalized retention strategies.

\newpage

% ---------------- PROBLEM ----------------
\section{Problem Statement}

Customer churn negatively impacts revenue and long-term sustainability. Manual identification of churn risk is inefficient.

\textbf{Objectives:}
\begin{itemize}
\item Predict customer churn
\item Identify key influencing factors
\item Provide actionable insights
\item Build an interactive system
\item Enable AI-driven recommendations
